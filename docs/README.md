# Tài liệu thiết kế

Nơi lưu tài liệu thiết kế cho hệ thống quản lý công việc/đội ngũ trên nền
NocoBase.

Gợi ý các tài liệu sẽ bổ sung dần:

- `data-model.md` — mô hình dữ liệu (collections/fields/relations) cho OKR,
  KPI, Task, Dự án, Strategy Hub.
- `workflows.md` — quy trình nghiệp vụ, phân quyền theo vai trò.
- `ui-flows.md` — luồng màn hình chính cho từng module.
- `decisions/` — các quyết định kiến trúc quan trọng (ADR).
